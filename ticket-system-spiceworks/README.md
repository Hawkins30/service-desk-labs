## Purpose
The purpose of this lab is to demonstrate hands-on experience using a ticketing system in a first-line IT support environment. The lab focuses on creating and managing tickets, differentiating between incidents and service requests, prioritising work, documenting troubleshooting steps, communicating with users, and closing tickets professionally.

This lab reflects real-world service desk workflows used in junior IT support and helpdesk roles.
